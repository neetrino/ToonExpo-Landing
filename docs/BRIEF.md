# Նախագծի տեխզադրանք — Toon Expo Landing

---

## Նկարագրություն

Շինարարական ցուցահանդեսի **կենտրոնացված հարթակ**՝ 70–100+ մասնակիցների (նախագծերի) համար։ Յուրաքանչյուրի համար ավտոմատ **լենդինգ** (`/company/{slug}`), գլխավոր էջ՝ **քարտեզ + ցանկ**։ Տվյալների աղբյուրը՝ **Excel/CSV** (`docs/data/ToonExpoData2026.*`) — **53 ակտիվ դաշտ**։

## Թիրախային լսարան

Ցուցահանդեսի այցելուներ (մոբայլ), ներդրողներ, գնորդներ։ Մասնակից ընկերություններ՝ ադմինով լրացնում են նույն դաշտերը, որոնք ցուցադրվում են լենդինգում։

## Հիմնական ֆունկցիաներ (առաջնայնություն)

1. **Լենդինգ նախագծի** — Figma մակետին համապատասխան, դաշտերը `DATA_FIELDS_SCHEMA`-ից — **բարձր**
2. **Պայմանական ցուցադրում** — դատարկ դաշտ/բլոկ չցուցադրել — **բարձր**
3. **Ադմին** — բոլոր 53 դաշտը յուրաքանչյուր մասնակցի համար — **բարձր**
4. **Գլխավոր** — քարտեզ + ցանկ + որոնում — **միջին**
5. **Excel/CSV ներմուծում** — **միջին**
